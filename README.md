## 技术结论摘要

- 推荐主路线：`Stage 模型 + UIAbility + ArkTS + ArkUI`
- 推荐存储方案：`Preferences + RDB`
- 推荐通信方案：`HTTP + WebSocket`
- React 可用，但仅建议作为局部 Web 增强方案，不建议作为主界面技术
