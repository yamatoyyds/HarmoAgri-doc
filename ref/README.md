# 智慧农业控制系统研究与策划索引

本目录用于沉淀针对 `doc/DOC_target.md` 的研究结论、技术策划、文档结构设计与开发计划。

## 文档清单

- `doc/ref/01_赛题解读与需求拆解.md`：对原始赛题要求进行结构化拆解，补全边界、角色、场景与交付物映射。
- `doc/ref/02_技术路线研究与框架选型.md`：分析 OpenHarmony 下的可用技术路线、现成框架、React 可行性与推荐选型。
- `doc/ref/03_从零开始构建应用方案.md`：说明从 0 到 1 搭建应用的步骤、模块划分、数据流与系统分层建议。
- `doc/ref/04_完整文档结构与交付规划.md`：给出建议的完整文档目录结构、阶段性交付件及输出标准。
- `doc/ref/05_开发计划与实施路径.md`：提供可执行的开发排期、里程碑、风险与验证策略。
- `doc/ref/06_结论摘要.md`：汇总最终建议，便于快速阅读与立项决策。
- `doc/ref/07_Windows主环境下OpenHarmony应用开发与调试手册.md`：在不考虑 WSL 的前提下，给出 Windows 单主环境的开发、联调、真机调试与交付参考手册。

## 使用建议

- 若目标是先完成方案评审，优先阅读 `doc/ref/06_结论摘要.md` 与 `doc/ref/04_完整文档结构与交付规划.md`。
- 若目标是进入实施阶段，继续阅读 `doc/ref/03_从零开始构建应用方案.md` 与 `doc/ref/05_开发计划与实施路径.md`。
- 若目标是直接落地 Windows 单环境开发，继续阅读 `doc/ref/07_Windows主环境下OpenHarmony应用开发与调试手册.md`。
- 若目标是确认技术方案是否适合 React 或现成框架，重点阅读 `doc/ref/02_技术路线研究与框架选型.md`。
