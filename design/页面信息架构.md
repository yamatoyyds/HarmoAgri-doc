# 页面信息架构

## 1. 页面树

```text
首页总览
├─ 环境卡片区域
├─ 设备状态区域
├─ 最新告警区域
└─ 快捷入口区域

设备页
├─ 设备列表
└─ 设备详情

灌溉页
├─ 土壤湿度卡片
├─ 灌溉建议
├─ 手动控制
└─ 灌溉记录

光照页
├─ 光照强度卡片
├─ 自动补光配置
├─ 手动亮度控制
└─ 补光记录

告警页
├─ 告警列表
├─ 告警详情
└─ 处理状态

设置页
├─ 网关设置
├─ 阈值设置
├─ 联动规则
└─ 扩展设备设置
```

## 2. 页面跳转关系

- 首页可进入设备页、灌溉页、光照页、告警页、设置页。
- 告警页可反向定位到相关设备或相关规则。
- 设置页可进入网关配置、规则配置和扩展设备管理子页。

## 3. 页面导航建议

### 3.1 一级导航建议

- 首页
- 设备
- 灌溉
- 光照
- 告警
- 设置

### 3.2 二级页面建议

- 设备详情页
- 告警详情页
- 阈值规则编辑页
- 联动规则编辑页
- 网关连接配置页
- 扩展设备配置页

## 4. 首页信息优先级

优先级从高到低：

1. 当前关键环境指标
2. 异常状态和告警提示
3. 在线设备概览
4. 快捷控制入口
5. 最近执行记录

## 5. 页面级信息结构说明

### 5.1 首页

| 区块 | 内容 | 目的 |
|---|---|---|
| 环境卡片 | 温度、湿度、光照、二氧化碳、土壤湿度 | 快速判断环境状态 |
| 异常摘要 | 当前告警数量、最高级别告警 | 快速感知风险 |
| 设备摘要 | 在线设备数、离线设备数 | 快速判断系统状态 |
| 快捷入口 | 灌溉、补光、告警、设置 | 降低操作层级 |

### 5.2 设备页

| 区块 | 内容 | 目的 |
|---|---|---|
| 设备列表 | 设备名称、类型、在线状态 | 总览所有设备 |
| 过滤区 | 按类型、状态筛选 | 快速定位设备 |
| 设备详情 | 当前状态、最近动作、操作入口 | 管理和控制设备 |

### 5.3 灌溉页

| 区块 | 内容 | 目的 |
|---|---|---|
| 指标区 | 土壤湿度、建议值 | 支撑灌溉决策 |
| 手动控制区 | 开启/停止、灌溉时长 | 手动操作 |
| 策略区 | 自动灌溉开关、阈值 | 自动化控制 |
| 记录区 | 最近灌溉记录 | 便于回顾 |

### 5.4 光照页

| 区块 | 内容 | 目的 |
|---|---|---|
| 指标区 | 当前光照强度 | 判断是否需要补光 |
| 控制区 | 开关、亮度等级 | 手动控制 |
| 自动区 | 阈值和目标亮度 | 自动补光 |
| 记录区 | 最近补光记录 | 便于回顾 |

### 5.5 告警页

| 区块 | 内容 | 目的 |
|---|---|---|
| 告警列表 | 时间、等级、内容、状态 | 快速查看告警 |
| 过滤区 | 按等级、状态筛选 | 快速定位 |
| 详情区 | 触发值、关联设备、处理状态 | 告警追踪 |

### 5.6 设置页

| 区块 | 内容 | 目的 |
|---|---|---|
| 网关配置 | 地址、端口、连接状态 | 管理连接 |
| 规则配置 | 阈值规则、联动规则 | 管理自动化 |
| 扩展配置 | 新设备接入参数 | 管理扩展能力 |

## 6. 页面状态说明

| 页面 | 状态 | 含义 |
|---|---|---|
| 首页 | loading | 首次加载中 |
| 首页 | ready | 展示正常 |
| 首页 | offline | 网关离线 |
| 设备页 | empty | 无设备数据 |
| 灌溉页 | submitting | 控制提交中 |
| 光照页 | auto_enabled | 自动补光启用 |
| 告警页 | filtered | 告警筛选中 |
| 设置页 | saving | 配置保存中 |
