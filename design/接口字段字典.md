# 接口字段字典

| 文档版本 | V1.1 |
|---|---|
| 创建日期 | 2026-03-15 |
| 更新日期 | 2026-03-16 |
| 文档作者 | OpenCode |

## 1. 文档目的

统一前端、网关、测试和文档中的字段命名、类型和含义，减少联调歧义。

## 2. 通用响应字段

| 字段名 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| code | number | 是 | 响应码，`0` 为成功 | `0` |
| message | string | 是 | 响应说明 | `ok` |
| requestId | string | 否 | 请求追踪标识 | `req-20260316-0001` |
| data | object/array | 是 | 业务数据 | `{}` |
| details | object | 否 | 错误附加信息 | `{ "deviceId": "pump-001" }` |

## 3. 通用业务字段

| 字段名 | 类型 | 说明 | 示例 |
|---|---|---|---|
| id | string | 主键或唯一标识 | `pump-001` |
| name | string | 展示名称 | `主灌溉水泵` |
| type | string | 类型枚举 | `PUMP` |
| timestamp | string | ISO 8601 时间字符串 | `2026-03-15T10:01:00Z` |
| requestId | string | 幂等请求标识 | `cmd-20260315-0001` |
| enabled | boolean | 是否启用 | `true` |
| status | string | 当前状态或结果状态 | `ON` |

## 4. 网关状态字段

| 字段名 | 类型 | 说明 |
|---|---|---|
| online | boolean | 网关是否在线 |
| lastHeartbeat | string | 最近心跳时间 |
| connectedDeviceCount | number | 已连接设备数量 |
| version | string | 网关版本 |

## 5. 实时环境数据字段

| 字段名 | 类型 | 单位 | 说明 |
|---|---|---|---|
| temperature | number | `°C` | 当前温度 |
| humidity | number | `%` | 当前湿度 |
| light | number | `lux` | 当前光照强度 |
| co2 | number | `ppm` | 当前二氧化碳浓度 |
| soilMoisture | number | `%` | 当前土壤湿度 |
| timestamp | string | - | 数据采集时间 |

## 6. 设备字段

| 字段名 | 类型 | 说明 |
|---|---|---|
| id | string | 设备唯一 ID |
| name | string | 设备名称 |
| type | string | 设备类型 |
| category | string | `SENSOR` / `ACTUATOR` |
| online | boolean | 设备是否在线 |
| status | string | 当前状态，如 `ON`、`OFF` |
| level | number | 档位或亮度等级 |
| location | string | 设备所属区域 |

## 7. 控制命令字段

| 字段名 | 类型 | 说明 | 示例 |
|---|---|---|---|
| command | string | 命令类型 | `TURN_ON` |
| params | object | 附加参数 | `{ "durationSec": 30 }` |
| requestId | string | 请求 ID | `cmd-20260315-0001` |
| accepted | boolean | 网关是否接收 | `true` |
| resultStatus | string | 执行结果状态 | `SUCCESS` |
| message | string | 结果说明 | `accepted` |
| deviceId | string | 目标设备 ID | `pump-001` |

## 8. 告警规则字段

| 字段名 | 类型 | 说明 |
|---|---|---|
| id | string | 规则 ID |
| name | string | 规则名称 |
| metricType | string | 指标类型，如 `TEMPERATURE` |
| operator | string | 比较符，如 `>`、`<` |
| thresholdValue | number | 阈值 |
| level | string | `WARNING` / `CRITICAL` |
| enabled | boolean | 是否启用 |

## 9. 告警事件字段

| 字段名 | 类型 | 说明 |
|---|---|---|
| id | string | 事件 ID |
| ruleId | string | 对应规则 ID |
| deviceId | string | 关联设备 ID |
| metricType | string | 告警指标类型 |
| level | string | 告警等级 |
| message | string | 告警说明 |
| status | string | `NEW` / `ACK` / `RESOLVED` |
| triggeredValue | number | 触发值 |
| createdAt | string | 触发时间 |
| resolvedAt | string | 恢复时间 |

## 10. 联动规则字段

| 字段名 | 类型 | 说明 |
|---|---|---|
| id | string | 联动规则 ID |
| name | string | 联动规则名称 |
| condition | object | 条件定义 |
| action | object | 动作定义 |
| enabled | boolean | 是否启用 |

### 10.1 `condition` 子字段

| 字段名 | 类型 | 说明 |
|---|---|---|
| metricType | string | 指标类型 |
| operator | string | 比较符 |
| thresholdValue | number | 阈值 |

### 10.2 `action` 子字段

| 字段名 | 类型 | 说明 |
|---|---|---|
| deviceId | string | 目标设备 |
| command | string | 执行动作 |
| params | object | 动作参数 |

## 11. 历史事件与分页字段

| 字段名 | 类型 | 说明 |
|---|---|---|
| type | string | `ALERT` / `COMMAND` / `RULE` |
| startTime | string | 开始时间 |
| endTime | string | 结束时间 |
| pageNo | number | 页码 |
| pageSize | number | 每页条数 |
| total | number | 总条数 |
| list | array | 当前页列表 |

## 12. WebSocket 推送字段

| 字段名 | 类型 | 说明 |
|---|---|---|
| topic | string | 推送主题 |
| timestamp | string | 服务端推送时间 |
| data | object | 推送载荷 |
