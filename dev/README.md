# 开发阶段文档索引

本目录用于沉淀智慧农业控制系统在实际开发阶段产生的规划、记录、问题修复与阶段结论文档。

## 目录约定

```text
doc/dev/
├─ README.md
├─ plan/
├─ log/
└─ fix/
```

## 子目录说明

- `plan/`：开发规划、原型方案、里程碑、技术预研、可行性评估。
- `log/`：按日期记录开发阶段结论、阶段决策、进展日志。
- `fix/`：问题修复记录、风险处置、经验沉淀。

## 当前阶段重点

- 目标：先完成适配 `10.1` 寸横屏触摸设备的 OpenHarmony 原型界面 demo。
- 范围：先做界面展示、交互路径、状态演示和占位数据，不做真实网关、设备协议和后端联调。
- 原则：先验证演示效果与信息架构，再逐步替换为真实服务实现。
